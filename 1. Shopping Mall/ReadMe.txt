: 고찰
    - 기능 구현 자체는 크게 어렵지 않았던 것 같은데, 여러 상품에 같은 검색어가 있을 경우 제일 첫 번째 상품만 나오고, 나머지는 나오지 않는 문제가 있었다.
      처음엔 불리언 타입의 변수를 둬서 실행 이전에 false로 둔 뒤, 첫 실행시에만 true로 변경하여 이후에 나머지도 출력하도록 하려했으나 문제는 이 부분이 아니었다.
    - 검색버튼을 클릭한 후의 반복문이 문제였다. for (let i = 0; i < products.length; i++) 반복적으로 카드를 생성하기 위한 구문인데, 초기에 length를 products(json자료)가 아닌 카드의 갯        수만큼 줬기 때문에, $(".container-templete").html(''); 로 인해서 카드의 내용이 아예 비워진 상태에서 시작할 때는 반복문이 동작하지 않았고, 위의 비우는 코드의 위치를 바꾸더라도             카드 하나를 만들고 초기화가 되기 때문에 한 장의 카드만이 출력되는 것이었다.
    - 고찰을 적으면서 느끼는데 내가 짠 코드더라도 설명하는게 참 어렵다는 것을 새삼 느꼈다. 내가 읽어도 무슨 말인지 하나도 모르겠어.. 코드를 작성/수정하면서 이 코드가 무엇이고 어떤 동작을 하       는지, 어떤 변수(함수)와 연결되는지를 상세히 기록해 놓아야 다른 사람이 내 코드를 읽었을 때 이해할 수 있을 것 같다.
